# Portfolio Website (Vue)

Portfolio website migrated from static HTML pages to a Vue + Vite app with Vue Router.

## Run Locally
1. Install dependencies:
   - `npm install`
2. Start development server:
   - `npm run dev`
3. Build production bundle:
   - `npm run build`

## App Structure
- `src/App.vue` - shared application shell (header, footer, theme, animations)
- `src/router/index.js` - route definitions
- `src/views/` - page views (overview, about, projects, extracurricular, cv, project details)
- `styles.css` - shared site styles
